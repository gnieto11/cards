Tech stack:

* Javascript
* Vue
* NPM
* Vue-router
* Axios
* Vuex
* Vuetify

Clone Project:
```
git clone https://github.com/gnieto11/login-front.git
```

Install dependencies:
```
npm install
```
Run:

```
npm run serve
```
