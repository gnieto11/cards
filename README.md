The function of this project is to be able to log in as a user, and that it can create its own tasks and reminders.

Tech stack:

* Javascript
* Vue
* NPM
* Vue-router
* Axios
* Vuex
* Vuetify

Demo: https://tarjetas.netlify.app/login

Clone Project:
```
git clone https://github.com/gnieto11/login-front.git
```

Install dependencies:
```
npm install
```
Run:

```
npm run serve
```
